# POKEDEX
An electronic encyclopedia about Pok√©mons with different sections: card, grid, compare, favorites.
You can search a pokemon by id or name.

### Card
Various information about pokemon: from its name to the chain of evolution.

### Grid
Section with a large number of pokemons on the screen

### Compare
Here you can compare the characteristics of pokemons.

### Favorites
All the pokemons you've licked appear here.

## Installation
```bash
git clone https://github.com/zronev/react-pokedex.git
cd react-pokedex
npm install
```

## Start & watch
```bash
npm start
```

## Simple build for production
```bash
npm run build
```
